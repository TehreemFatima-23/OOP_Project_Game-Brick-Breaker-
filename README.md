# Brick Breaker Game (JavaFX)

This was my final project for the **Object Oriented Programming (OOP)** course in Semester 2. 

### Tech Stack

**Language:** Java 
**GUI Framework:** JavaFX / FXML 
**IDE:** IntelliJ IDEA

### Key Features

**Collision Detection:** Smooth physics between the ball, paddle, and bricks.
**Level Management:** Increasing difficulty as the player progresses.
**Sound Effects:** Integrated audio for bounces and brick breaks.
**High Score Tracking:** Saves the best scores using file handling.

### OOP Concepts Used

**Encapsulation:** Used for protecting the user's data and state of the game.
**Inheritance:** Game entities share common properties from a base class.
**Abstraction:** Managed complex game loops through simplified interfaces.
**Interfaces:** Custom interfaces were implemented for game logic.
