# OOP_Project_Game-Brick-Breaker
A desktop Brick Breaker game developed using Java, JavaFX and Object Oriented Programming principles. 

Brick Breaker game (using JavaFX)

       This was the game as final project for the course Object Oriented Programming in semester 2. It is a classic Brick breaker game built using Java and JavaFX framework for GUI.

Language used: Java
GUI Framework: JavaFX / FXML
IDE used: IntelliJ IDEA

Key Features:

Collision Detection: Smooth physics between the ball, paddle, and bricks.
Level Management: Increasing difficulty as the player progresses.
Sound Effects: Integrated audio for bounces, brick breaks, getting power up and power effects.
High Score Tracking: Saves the best scores using file handling.

OOP concepts used:

Encapsulation: Uesd for protecting the user's data and state of the game.
Inheritance: Game entities share common properties from a base class.
Abstraction: Managed complex game loops through simplified interfaces.
Interfaces are also used.
